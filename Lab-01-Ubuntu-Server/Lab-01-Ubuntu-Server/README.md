# Lab 01 - Instalação do Ubuntu Server 26.04 no VirtualBox

## Objetivo

Criar um servidor Linux para servir como base do laboratório de Cloud Computing e Cybersecurity.

## Ambiente

* Host: Windows 11
* VirtualBox
* Ubuntu Server 26.04 LTS
* Memória da VM: 4 GB
* Processadores: 2 vCPUs
* Disco: 40 GB
* Firmware: UEFI
* Rede: NAT

## Ferramentas utilizadas

* VirtualBox
* Ubuntu Server 26.04
* OpenSSH Server

## Passo a passo

* Criação da máquina virtual.
* Configuração de memória e processadores.
* Configuração do disco virtual.
* Instalação do Ubuntu Server.
* Configuração do usuário.
* Instalação do OpenSSH Server.
* Correção do layout do teclado.
* Primeiro acesso ao sistema.

## Validação

Após a instalação, foi possível:

* Inicializar o Ubuntu Server.
* Realizar login com o usuário criado.
* Acessar o terminal Linux.
* Confirmar o funcionamento do sistema.

## Aprendizados

Durante este laboratório foram praticados:

* Criação de máquinas virtuais.
* Instalação de sistemas operacionais.
* Configuração inicial de servidores Linux.
* Conceitos básicos de virtualização.
